# EventConnect - Final Project (WDD 330)

## What this includes
- Event browsing via Ticketmaster Discovery API
- Event detail view with Mapbox map
- Create events stored locally (localStorage)
- Favorites stored in localStorage
- Ratings & reviews (localStorage per event)
- Toast notifications
- Dark mode (persisted)
- Responsive and improved styles
- Dashboard for created events, favorites, last viewed and reviews
